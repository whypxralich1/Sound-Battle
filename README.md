# BeatBattle (временное название)

## Описание

Сервис-игра для развлечения/совершенствования навыков в сфере электронной музыки

## Функциональность

- Возможность соревноваться с другими игроками/практиковаться в одиночку
- Другие виды интерактивных "тренажеров"
- Возможность общения с другими пользователями платформы
- Возможность совместного написания музыки
## Запуск

Открытие платформы предварительно запланировано на вторую четверть 2026 года
